Exercise 1

One way to clean incoming data is to remove any errors or unnecessary spaces.

Write a function that will be given a value that is a string and then return it with all spaces removed.

Next create a form that has a label and a <textarea></textarea> for the user to enter data.

The form will also have a submit button which will process the text from the textarea, and display the results in an output area underneath the form.

Exercise 2

In this exercise you will be creating a javascript program that creates an array of user entered words.

To do so follow these steps:

  1. Create a form with a label, text input and button
  2. The label will read "Enter Word", the text inout will be named "word", and button will display the "add" and be a submit type button.
  3. When the user enters a word, and presses enter, the form will NOT refresh the page.
  4. Instead the javascript will check to see if the word is blank, and if so nothing is done.
  5. If a word has been entered the JavaScript will then add the word to an array
  6. After adding the word to the array, the Javascript then selects the whole word in the word field ready for the user to type in the next word.
  7. Finally, the javascript will update the output area by adding each new word to a list, with the most recent shown first.
Example of the word list output
  • banana
  • zebra
  • apple

When a new word, "poppy" is added, the output will be:

  • poppy
  • banana
  • zebra
  • apple
