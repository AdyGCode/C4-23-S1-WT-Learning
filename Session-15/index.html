Exercises

Complete each of the exercises from Session 14, and the following exercises.

Exercise 1: Faster Counter

Update the counter to execute every ONE second instead of every 2.5 seconds.

Exercise 2: Date for a Clock

This exercise asks you to create the first part of a clock.

In the class we looked at the use of setTimeout, setInterval and the Date object.

Using this knowledge, and the provided HTML in this file, write the JS to:

  • Add the Date to the clock in the form "Wednesday 1 May"
  • Do not add the st, nd, rd, th to the date number.

Exercise 3: Making date be 1st, 2nd, 3rd, 4th, etc

Continuing your clock exercise, we now want the date to have the correct ordinal indicator.

Create a function myOrdinal which is given an integer, and it will output a string with the number and its ordinal.

1st, 2nd, 3rd, 4th, 5th, ... 11th, 12th, 13th... 21st, 22nd, 23rd, ...

  • Hint: https://www.sitepoint.com/convert-numbers-to-ordinals-javascript/

Exercise 4: Add the Time

Continuing your clock exercise.

Now we want to display the time on the clock.

Begin by writing the code to add the time in the form HH:MM:SS to the clock face.

The time will update every 0.5 seconds.

Exercise 5: Remove Seconds and Flash the :

Final Step for the clock.

Modify the display by removing the seconds.

Instead of showing the seconds, the clock will now show and hide the : between the hours and minutes once every 0.5 seconds.
